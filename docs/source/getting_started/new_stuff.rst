What's new in Cool-chic 5.0?
============================

This release focuses on image coding, implementing the contributions presented
in the following paper: `Cool-chic 5.0: Faster Encoding and Inter-Feature
Entropy Modeling for Overfitted Image Compression, Ladune et al.
<https://arxiv.org/abs/2605.02726>`_.

* **10x encoding speed-up** versus Cool-chic 4.2

* Better image compression: **Rate decrease of -10%** versus Cool-chic 4.2 for identical quality

* Low decoding complexity of **2000 MAC / pixel**

Check-out the `release history
<https://github.com/Orange-OpenSource/Cool-Chic/releases>`_ to see previous
versions of Cool-chic.
