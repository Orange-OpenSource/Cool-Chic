Format
======

The ``format`` submodule gather the different functions used to read and write
images and video frames with different formats.

   * ``yuv.py`` for `YUV data <https://en.wikipedia.org/wiki/Y%E2%80%B2UV>`_ files.

   * ``png.py`` for PNG data

   * ``ppm.py`` for `PPM data <https://en.wikipedia.org/wiki/Portable_pixmap>`_

   * ``data_type.py`` defines some useful types.
